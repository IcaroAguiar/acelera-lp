# Acelera Conexoes - Landing Page

Repositorio da landing page do Acelera Conexoes, projeto Star Agency.

## Materiais iniciais

- `docs/ACELERA_LP_BRIEFING_EXTRAIDO.md`: briefing estruturado para conteudo, copy, requisitos tecnicos, formulario, ativos e lacunas.
- `docs/DESIGN.md`: contrato inicial de design para orientar variantes visuais minimalistas, premium e focadas em conversao.
- `source/Acelera_LP_Brief.pdf`: PDF original usado como fonte primaria.

## Direcao

A LP deve qualificar leads para uma experiencia premium de pilotagem em Interlagos. A conversao principal e o envio do formulario de interesse; nao e checkout nem venda direta de ingresso.

Antes de implementar, confirmar:

- grafia oficial da marca;
- dominio;
- numero do WhatsApp;
- Pixel Meta;
- GA4/GTM;
- destino do formulario;
- ativos fotograficos reais.
